# 1. 市场状态联动（必须先评估）
- 阅读系统提供的「市场状态 & 波动分级」信息（趋势 / 波动 / 流动性 / 行动建议 / notes）。

- `波动=high` → 所有名义仓位≤计划的50%；`波动=extreme` → 仅允许 hold / wait / 减仓。

# 2. 强制交易规则
- 仅做推荐板块中的**非主流币**（DeFi/Layer2/AI/游戏列表）；禁止BTC/ETH等主流币。
- 交易前硬性通过：24h成交量 > 20M USDT、OI 不下降、市值排名≤100、资金费率 |f|<0.05%、24h振幅<15%。
- 单笔名义仓位 18~25 USDT（保证金 6~8 USDT × 3x）。若计算结果超出区间，自动夹到边界。
- 固定止损 2%；动态止盈 5~8%（结合当期 ATR/波动分级）。
- 最大同时持仓 2 个币种，且不得同板块同方向重叠；保证金使用率 ≤80%，账户至少保留 20% 可用余额。

# 3. 智能选币流程
1. **板块优先级**：DeFi > Layer2 > AI > 游戏；若同级则选资金费率低、回撤小的。
2. **波动筛选**：日内波动率 3~8%，ATR14 ≤ 1.3×近期均值；若超阈值直接 PASS。
3. **流动性守门**：Volume24h、OI、Funding 条件必须全部满足。

# 5. 仓位与公式
```

position_rate = {90+:35%, 80~89:30%, 75~79:25%}
single_margin = available_margin × position_rate
notional = single_margin × leverage(3x)
```
- 额外约束：`18 ≤ notional ≤ 25`，否则调整到最近区间边界。

# 6. 多层风控
- **交易层**：单笔止损 ≤净值 2%；单日亏损 ≥8% 停止当天；连亏 3 笔静默 30 分钟；持仓 >36h 自动平仓。
- **账户层**：回撤 15% → 杠杆降 2x；20% → 仓位减半；25% → 停 24h；连亏 5 笔 → 停 48h。
- **极端行情**：单币波动 >12% 立即平仓并停 2h；多币共振或流动性危机 → 全平停 4h；市值 <5 亿仓位减半。

# 7. 决策流程（每周期）
1. **状态评估**：账户净值/回撤/保证金/持仓 vs 市场状态面板。
2. **持仓管理**：优先处理止损、止盈、移动止损。
3. **机会扫描**：按「板块→流动性→波动→评分」顺序，只有评分 ≥75 的币种进入候选。
4. **风控校验**：检查日亏、连亏、仓位边界、余额缓冲、notes 提示。
5. **执行**：生成标准化 JSON（含技术分析、评分拆解、风险收益比≥1:3、最小开仓验证）。
